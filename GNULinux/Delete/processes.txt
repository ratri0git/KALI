

TOP :

    SUMMARY DATA AREA :

        INTERACTIVE DETAILS OF ALL ONGOING PROCESSES : top

        HIDE TOP MOST LINE IN VIEW : press l

        SEE INFO FOR ALL CPUS : press 1

        CPU USAGE GRAPH : press t

        RAM AND SWAP GRAPH : press m

        CHANGE MEMORY UNITS KB MB GB TB : press E

    COLUMN HEADERS AND TASK INFO PROCESSE LIST : 

        CHANGE REFRESH RATE OF TOP IN SECONDS : press d OR spacebar or enter

        FILTER BY USERNAME : press u and UserName     


NICE : 

    CHANGE NICE VALUE : press r and PID 
    sudo renice -n Value -p PID 
    sudo nice -n Value Commands


 

   
FOREGROUND AND BACKGROUND : 

    RUN JOB IN BACKGROUND : commands &

    RESUME STOPPED JOB IN FOREGROUND : fg

    RESUME STOPPED JOB IN BACKGROUND : bg

    LIST JOBS and PID : jobs -l


NOHUP : 

    RUN JOB IN BACKGROUND AND DIRECT OUTPUT TO NOHUP : nohup Commnads
